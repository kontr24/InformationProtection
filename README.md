# InformationProtection
4 курс, Защита информации.  
Шифры:  
- шифр Атбаш;
- шифр Цезаря;
- квадрат Полибия;
- прямоугольник Плейфера;
- шифр Виженера;
- аффинный шифр;
- шифр перестановок;
- шифр гаммирования.  
(Лабораторные работы. Язык - С#, интерфейс - Windows Forms)
